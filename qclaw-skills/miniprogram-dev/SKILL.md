# 小程序开发管理 - SKILL.md

## 技能简介

管理微信小程序开发的自主技能。可以自动从微信开发者工具上传代码、管理版本、操作小程序配置等。

## 核心能力

### 1. 自动上传代码

**触发方式：**
- 用户说「上传」「上传小程序」「上传代码」
- 检测到用户在开发小程序

**上传流程（自主探索）：**

```
第1次尝试 → 检测微信开发者工具窗口 → 打开上传对话框 → 填表 → 提交
第2次尝试 → 如果失败，换一种交互方式（键盘事件/鼠标/菜单）
第3次尝试 → 如果还失败，检查是否有错误弹窗 → 尝试解决
第N次尝试 → 记录失败模式，尝试新方法
只有全部失败后才询问用户
```

### 2. 版本管理

- 查看当前版本状态
- 切换体验版/开发版
- 查看审核状态
- 发布版本（需要确认）

### 3. 测试二维码

- 生成体验版二维码
- 打开二维码页面

### 4. 状态检查

- 检查上传状态
- 检查审核状态
- 检查版本信息

## 自动行为规则

### 自主探索原则

1. **遇到问题先自己尝试3次以上**
   - 换不同的方法尝试
   - 记录每次尝试的结果
   - 从失败中学习

2. **不要轻易让用户手动操作**
   - 除非真的无法自动完成
   - 可以问「要我继续尝试其他方法吗？」

3. **只询问关键确认**
   - 「要开始上传吗？」
   - 「确认发布这个版本吗？」
   - 而不是每一步都问

### 错误处理

- 遇到错误弹窗 → 自动尝试关闭或处理
- 上传失败 → 记录错误，尝试下一个方法
- 窗口找不到 → 尝试激活或打开微信开发者工具

## 使用示例

| 用户说 | 技能行为 |
|--------|----------|
| 「上传小程序」 | 自动开始上传流程，只需要最后确认 |
| 「看看版本状态」 | 自动检查并报告当前版本情况 |
| 「生成二维码」 | 自动打开体验版二维码页面 |
| 「帮我发布」 | 检查条件，询问确认后执行 |

## 技术实现

### 窗口检测

```powershell
# 检测微信开发者工具
Get-Process -Name "wechatdevtools" -ErrorAction SilentlyContinue
```

### 上传对话框交互

通过以下方式自动化：
1. 菜单操作（Alt+F → U）
2. 快捷键
3. 键盘事件模拟
4. 窗口消息

### 版本管理

通过 mp.weixin.qq.com 网页操作或 API

---

## 注意事项

1. 微信开发者工具必须保持登录状态
2. 小程序需要有 AppID 才能上传
3. 上传前确保代码没有严重语法错误
4. 发布需要先通过审核
