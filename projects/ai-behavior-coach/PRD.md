# AI行为教练 App - 产品调研文档

> 创建时间：2026-04-14
> 状态：调研阶段

---

## 一、产品概念

### 核心定位
**自动行为数据采集 + AI 教练分析 + 闭环反馈**

### 一句话描述
一个知道你每天真正在做什么，并且能给你精准改进建议的 AI 私人教练。

### 核心价值
1. 大多数人的问题不是"不知道怎么做"，是"不知道自己实际在干嘛"
2. 数据+AI分析，把你从"我以为"的黑洞里拽出来，给你一个客观的镜子
3. 信息不改变人，重复决策才改变人

---

## 二、用户痛点

| 痛点 | 描述 |
|------|------|
| 自我认知偏差 | 以为自己很努力，实际在刷手机3小时 |
| 缺乏客观反馈 | 没人告诉你真相，或者说你自己不想看真相 |
| 执行力不足 | 知道该做什么，但没人督促，容易放弃 |
| 教练成本高 | 真人教练贵，不可能24小时在线 |
| 工具碎片化 | 追踪工具、AI工具、习惯工具各自独立，没有打通 |

## 三、市场竞品分析

### 已有产品

| App | 功能 | 优势 | 短板 |
|-----|------|------|------|
| RescueTime | 自动追踪电脑/手机使用 | 数据准确、自动化 | 只有统计，没有教练指导 |
| Forest | 专注时种树 | 游戏化激励 | 无行为分析，需手动启动 |
| Habitica | 习惯养成RPG | 有趣 | 需手动记录，无自动采集 |
| Coach.me | 真人教练+习惯追踪 | 人类教练质量高 | 成本高，不能高频互动 |
| Replika/Pi | AI聊天陪伴 | 情感支持 | 不追踪实际行为数据 |
| Rize/Timing | Mac时间追踪 | 细粒度数据 | 只有追踪，无教练功能 |
| 屏幕使用时间/数字健康 | 系统原生 | 无需安装 | 只有数据，无分析建议 |
| Notion AI/ChatGPT | AI对话分析 | 灵活强大 | 需手动输入数据 |

### 市场空白

**目前没有产品做到**：自动行为采集 + AI教练分析 + 闭环反馈

现有产品要么只有数据，要么只有教练但需手动输入，要么只有游戏化但没有智能分析。

## 五、核心功能模块

### 1. 数据采集层
- 屏幕使用时间（iOS/Android API）
- APP 使用频率和时长
- 位置/运动数据（可选）
- 文档编辑/浏览记录（可选，需权限）
- 自定义数据源接入

### 2. AI 教练层
- **行为分析**：识别时间分配模式、逃避行为、拖延模式
- **问题诊断**：找出阻碍进步的核心原因
- **方案生成**：个性化训练计划、每日任务
- **高频反馈**：每日复盘提问、周末汇总报告

### 3. 反馈交互层
- 微信/APP 推送提醒
- 对话式交互（自然语言问答）
- 数据可视化仪表盘
- 进度追踪和里程碑

## 六、MVP 开发路线

### Phase 1：手动输入 + AI 教练（最快验证）
- 用户手动截图或输入屏幕使用时间
- AI 分析并生成建议
- 定期提醒用户反馈
- **周期**：1-2 周
- **目标**：验证 AI 教练的核心价值

### Phase 2：小程序/轻量 App
- 一键导出屏幕使用时间数据
- 自动发送到 OpenClaw webhook
- 收到 AI 反馈推送
- **周期**：4-6 周
- **目标**：降低用户输入成本

### Phase 3：完整 App
- 自动采集、自动上报、自动反馈
- 多数据源整合
- 个性化教练训练模型
- **周期**：3-6 月
- **目标**：完整产品体验

## 七、技术实现路径

| 路径 | 难度 | 说明 |
|------|------|------|
| iOS 屏幕使用时间 API | 中等 | iOS 15+ 开放部分接口，需用户授权 |
| Android UsageStatsManager | 低 | Android 原生支持，权限较开放 |
| 第三方 SDK 集成 | 低 | 接入 RescueTime/Toggl 等 API |
| 自建采集层 | 高 | 监听系统广播，各厂商限制多 |

## 八、AI 教练 Prompt 模板

```
我想成为______的人（具体描述）。
我现在的状态是______（客观描述）。
约束条件：每天______时间可用。

请做我的私人教练：
1. 分析我提供的行为数据，识别问题模式
2. 生成一份训练计划，分阶段，每个阶段有明确里程碑
3. 每天给我一个当日任务（不超过30分钟）
4. 每天晚上问我三个问题：
   - 今天做了什么？
   - 遇到什么障碍？
   - 我可以怎么改进？
5. 如果我连续两天没有行动，要严厉问我："你是不是放弃了？"
6. 每周末复盘，调整下周计划

我承诺：认真对待每一次回答，不糊弄自己。
```

## 十、风险与边界

### 风险
1. **隐私担忧**：用户可能不愿意分享行为数据
2. **平台限制**：iOS/Android 对后台数据访问有限制
3. **执行依赖**：工具再好，用户不执行也白搭

### 边界
- AI 教练不能代替心理健康治疗
- 数据采集需要用户明确授权
- 产品只能提供镜子，最终改变靠用户自己

## 十一、核心洞察

> **金句**：你以为你知道自己是谁，但数据告诉你，你不是。

> **真相**：收集数据不行动 = 看着体检报告继续吃炸鸡。知道和做到之间，隔着一道真正的沟。这道沟数据帮不了你，只有你自己能跨过去。
