## 核心能力

### 1. 心理策略驱动
不是一套话术对所有人。教练会识别你的：
- **恐惧类型**（怕被拒绝？怕做不好？）
- **动力类型**（自驱型？外驱型？）
- **压力反应**（崩溃型？逃避型？）

然后**自动调整**任务推荐顺序和对话节奏。

### 2. 阶段导航
```
探索期 → 验证期 → 稳定期 → 扩张期
```
每个阶段有明确的目标和跃迁条件。教练知道你在哪个阶段、该做什么。

### 3. 障碍专项突破
如果同一个障碍出现3次，教练会停下来：
> "我发现你这周已经3次卡在同一个地方。我们专门处理这个，而不是一直绕过去。"

### 4. 成功模式自动提取
每次成交后，教练自动分析：
- 什么开场白有效？
- 什么内容转化最好？
- 哪个平台最适合你？

形成你的**个人成功模式库**。

### 5. 经验值 gamification
```
Lv1 新手(0 EXP)
    ↓
Lv3 验证者(300 EXP) → 收到第一笔钱
    ↓
Lv5 高手(1000 EXP) → 月收入稳定
    ↓
Lv7 宗师(2500 EXP) → 产品化/规模化
```

## 怎么启动

**第零步：定义成功**
> 教练："在你开始之前，先问你一个问题——你做这件事，成功的样子是什么？"

你需要回答：
- 成功定义（例：月入5000、有3个稳定客户）
- 时间线（例：3个月内）

**第一步：诊断3问**
1. 朋友经常向你请教什么？（= 你已能收费的）
2. 你每天什么时间有空？
3. 你愿意露脸还是幕后？

**第二步：制定第一个 Sprint**
- 3-7天验证期
- 2-3个任务，每任务拆成2-3步
- 每步 ≤ 10分钟

**第三步：执行**
- 教练每次问："你现在有多少时间？"
- 推荐最小行动
- 完成后记录，继续推进

## 常见问题

**Q：教练会帮我写内容吗��**
A：不会。教练帮你**想清楚**要做什么，但**你来做**。教练只逼你交付，不代工。

**Q：需要多少钱？**
A：Skill 本身免费。你需要花的只有：
- 时间（每天15-30分钟）
- 执行（去行动）

**Q：多久能看到效果？**
A：第一个 Sprint（3-7天）内完成第一笔交易的可能性很高，取决于你投入的时间。
