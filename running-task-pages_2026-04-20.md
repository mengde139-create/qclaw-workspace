# 2026-04-20 跑步专属页面开发

## 任务目标
按照冥想页面的思路，为跑步任务创建专属页面，替换通用 task/submit/result 页面。

## 完成内容

### 新增页面（4个）

| 页面 | 路径 | 用途 |
|------|------|------|
| 跑步设置 | `pages/running/running` | 选择距离(3/5/8/10km)、配速(慢跑/中速/快跑)、周期(7/14/30/90天) |
| 跑步指导 | `pages/running-task/running-task` | 跑前热身5步、跑步要点5条、安全提醒5条 |
| 跑步执行 | `pages/running-submit/running-submit` | 计时器、距离追踪、配速计算、卡路里估算、手动调距 |
| 跑步结果 | `pages/running-result/running-result` | 核心数据、详细统计、跑后拉伸5步、成就系统 |

### 修改文件

| 文件 | 修改内容 |
|------|----------|
| `app.json` | 添加4个跑步页面路由 |
| `index.js` | switchGoal 函数用 dedicatedPages 映射表统一处理专属页面跳转 |
| `goal.js` | 跑步类型跳转到 running 设置页 |

### 导航流程

```
首页
  ├─ 跑步（无目标）→ running 设置页 → 创建目标 → running-task 指导页
  └─ 跑步（有目标）→ running-task 指导页 → running-submit 执行页 → running-result 结果页
```

### 跑步设置页参数
- **距离**：3km(轻松慢跑) / 5km(日常训练) / 8km(进阶挑战) / 10km(长跑耐力)
- **配速**：慢跑(7-8分/km) / 中速(6-7分/km) / 快跑(5-6分/km)
- **周期**：7天 / 14天 / 30天 / 90天

### 跑步指导页内容
- 🔥 跑前热身：原地高抬腿→开合跳→弓步拉伸→脚踝环绕→慢走过渡
- 🏃 跑步要点：身体前倾、手臂摆动、呼吸节奏、步幅控制、不适即停
- ⚠️ 安全提醒：饮食、跑鞋、天气、交通、跑后放松

### 跑步执行页功能
- 实时计时器（分:秒）
- 距离追踪（基于配速估算，可手动±0.5km调整）
- 目标进度条
- 配速/卡路里/平均配速实时数据
- 开始/暂停/完成控制

### 跑步结果页内容
- 核心数据（距离）
- 详细数据（用时、消耗、平均配速）
- 跑后拉伸5步指引
- 累计统计（连续天数、累计公里、累计消耗）
- 成就系统（7天/30天/50km/100km）
