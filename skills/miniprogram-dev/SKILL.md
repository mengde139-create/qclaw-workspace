# 小程序开发管理 - SKILL.md

## 技能简介
管理微信小程序开发的自主技能。可以自动从微信开发者工具上传代码、管理版本、操作小程序配置等。

## 触发关键词
- 小程序
- 微信小程序  
- mini program
- 上传小程序
- 小程序上传
- 版本管理
- 体验版
- 发布小程序
- mp.weixin.qq.com
- 微信开发者工具

## 核心能力

### 1. 自动上传代码
触发方式：用户说「上传」「上传小程序」「上传代码」
上传流程（自主探索）：
- 第1次尝试 → 检测窗口 → 打开上传对话框 → 填表 → 提交
- 第2次尝试 → 换交互方式（键盘/鼠标/菜单）
- 第3次尝试 → 检查错误弹窗 → 尝试解决
- 只有全部失败后才询问用户

### 2. 版本管理
- 查看当前版本状态
- 切换体验版/开发版
- 查看审核状态
- 发布版本（需要确认）

### 3. 测试二维码
- 生成体验版二维码

### 4. 状态检查
- 检查上传状态
- 检查审核状态

## 自动行为规则
1. 遇到问题先自己尝试3次以上
2. 不要轻易让用户手动操作
3. 只询问关键确认（要开始吗？确认发布吗？）

## 技术实现
窗口检测：Get-Process -Name "wechatdevtools"
上传对话框：通过菜单(Alt+F -> U)或快捷键自动化

## 注意事项
1. 微信开发者工具必须保持登录
2. 需要 AppID 才能上传
3. 发布需要先通过审核
